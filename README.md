# Інструкція для роботи з проектом

## **Команди**

```bash
dotnet build Build.proj -p:Solution=Lab1 -t:Build
dotnet build Build.proj -p:Solution=Lab1 -t:Run
dotnet build Build.proj -p:Solution=Lab1 -t:Test

```
